# Privacy Policy

Free X Video Saver does not sell user data and does not collect browsing history.

The extension runs on `x.com` and `twitter.com` so it can detect videos on pages where the user chooses to use it. It may read the current page URL to copy or open the selected post link. Direct media URLs are sent only to Chrome's built-in downloads API so the browser can start a user-requested download.

The extension opens `https://jxsite.github.io/xdownload/` as a fallback when a direct downloadable media URL is not available on the page. Any data handling by that website is governed by that website's own privacy policy.

Permissions:

- `activeTab`: identify the active supported page after user interaction.
- `downloads`: start a user-requested video download.
- `storage`: reserved for future local settings; no personal data is stored in this version.
- Host access for `x.com`, `twitter.com`, and `video.twimg.com`: detect videos and download public media files from supported pages.
